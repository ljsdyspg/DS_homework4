实验任务
实验4（搜索）
【实验题目】
简单的学生信息管理系统的实现。
【实验目的】
学习使用散列表解决应用问题的方法。
【实验内容】
设有学生情况记录表，每个记录有下列数据项：学号、姓名、性别、籍贯。现使用散列表表示该学生情况记录表，并采用双散列法解决冲突。
所设计的系统重复显示以下菜单项。
 I：从一个文本文件中输入各记录来建立散列表。
 F：搜索并显示给定学号的学生记录。
 A：插入一个记录。
 D：删除一个记录。
 P：屏幕显示学生情况记录表。
 S：保存学生情况记录表。
 X：退出。
【提示】
    文件中每个记录占一行，以回车符分割，记录中的数据项以空格或逗号分割；假定文本文件可全部读入内存的记录数组中，保存学生情况记录表是将内存中的记录数组再次写回文本文件。
